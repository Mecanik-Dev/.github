# Mecanik Dev Ltd

Software development and security consultancy based in London, UK.

We've been building software and securing systems for over 20 years. Our work spans native desktop applications in C++/Qt, application and web security, penetration testing, server hardening, COBOL migration, AI/LLM integration, website development, and SEO auditing.

If it needs to be built, secured, modernised, or audited, that's what we do.

## Services

We offer custom software development with a focus on cross-platform C++/Qt desktop applications. On the security side, we handle penetration testing, vulnerability assessments, and server hardening.

For organisations stuck on legacy systems, our commercial tool [Easy COBOL Migrator](https://mecanik.dev) automates transpilation of COBOL codebases to C++, Python, Rust, Go, Java, and C#.

We also help businesses integrate AI and LLM capabilities into their products, build and redesign websites with a focus on performance and security, run in-depth SEO and technical audits, and provide pay-per-use REST APIs for developers through our platform.
